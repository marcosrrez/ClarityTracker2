  - POST /api/auth/client-signup  (line 5923-5970)
  - POST /api/auth/client-login   (line 6038-6078)

CURRENT CODE (VULNERABLE):
  Signup:  hashedPassword: password,              // ❌ PLAINTEXT
  Login:   if (client.hashedPassword !== password) // ❌ PLAINTEXT COMPARISON

FIXED CODE (SECURE):
